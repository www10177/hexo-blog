<!-- more -->
#### Panel

Toggle Sidebar : `Ctrl-k` +`Ctrk-b`  
Focus on Sidebar : `Ctrl-0`  

#### Beautified

Json Indent : `Ctrl-Alt-j`  
Html,CSS,JS (vue): `HTMLPrettify` (`Ctrl-Shift-H`)

#### Preview

Markdown : `Alt-M`

#### Emmet

Select Element : `ctrl-shift-a`  
Wrap Element `ctrl-shift-g`
